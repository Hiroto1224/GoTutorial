## コマンド一覧

| コマンド       | 使用箇所・用途               |
|------------|-----------------------|
| go run .   | go.modファイルの存在するディレクトリ |
| go test    | テストファイルのあるディレクトリ      |
| go test -v | テストの経過が確認できる          |
| go build   | 実行可能なバイナリファイル作成       |
